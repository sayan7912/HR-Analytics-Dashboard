# HR Analytics Dashboard (Tableau)

## Overview
Built an interactive HR Analytics Dashboard in Tableau to analyze employee attrition patterns and identify key factors affecting workforce retention.

## Tools Used
- Tableau
- Excel/CSV

## Key Metrics
- Total Employees: 1470
- Attrition Count: 237
- Attrition Rate: 16.12%
- Average Age: 37

## Key Insights
- Employees aged 25–44 account for the highest attrition.
- Sales department shows the highest attrition rate relative to workforce size.
- R&D records the highest attrition count but has a lower attrition rate than Sales.
- Sales employees exhibit lower job satisfaction levels, indicating a potential link to turnover.
- Male employees account for a larger share of attrition.

## Features
- Interactive Dashboard
- KPI Cards
- Action Filters
- Department-wise Attrition Analysis
- Age Group Analysis
- Education Field Analysis
- Job Satisfaction Analysis
